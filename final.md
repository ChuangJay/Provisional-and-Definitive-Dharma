##### **戊初、略標**

| ----- | ----------------------------------------------------------- | ------------------------------------------------------- |
| 戊初、略標 | 《解深密經》云：「勝義生，當知我依三種無自性性密意，說言一切諸法皆無自性。所謂相無自性性，生無自性性，勝義無自性性。」 | 佛親自說明：我說「一切法無自性」，**是依三種無自性而說**，並非單一否定。這一句是全論最根本的「判準宣告」。 |

|       | 答亦次第答彼二義。                                                   | 佛的回答也是**對應這兩個問題**，不是只回答其中之一。                            |
|       | 釋初義時，謂色乃至一切種智無量無邊諸法差別，說彼一切無自性者，攝為三種無自性中。                    | 首先回答「依何密意」：不論是色法、心法，乃至一切種智，**一切法的無自性都被總攝為三無自性**。        |
|       | 由宣說彼無自性理，易於了解，故總攝為三無自性。                                     | 佛將無量無邊的法差別，**方便攝為三類**，使眾生容易理解與修學。                       |
|       | 彼三亦攝勝義世俗一切法故。                                               | 這一句極關鍵：**三無自性同時涵蓋世俗與勝義一切法**，並非只限世俗。                     |
|       | 必當如是解者，以《般若經》等於五蘊、十八界、十二處、一切諸法，一一皆說無事、無性、無體。                | 若不如此理解，則《般若經》中對五蘊、十八界、十二處「一一說無性」將無法成立。                  |
|       | 尤說空性、法界、真如等勝義，一切異門皆無自性。                                     | 般若經不但說世俗法空，**連空性、法界、真如等勝義名相，也說無自性**。                    |
|       | 如是諸經宣說諸法無自性之法中，說無勝義，誰有心者能作彼說。                               | 結論：若說「般若經只說世俗空、不說勝義空」，是**全然站不住腳的解讀**。                   |

|      | 「何以故？此由假名安立為相，非由自相安立為相，是故說名相無自性性。」 | 理由在於：遍計相只是**依名言假立**，不是由法自身的自相成立。         |
|      | 此初二句，問答徧計所執即相無自性。                  | 經文已明確指出：**遍計所執＝相無自性所遮之對象**。              |
