# 六、這種寫法的三個關鍵優點

1. **學術可讀性高**

   * 標題是正式科判
   * 圖只是輔助，不喧賓奪主

2. **工具穩定**

   * 不怕 Mermaid 渲染崩壞
   * 不怕 PDF / GitHub / VS Code 不支援深巢狀

3. **後續可維護**

   * 新增一段＝新增一個小圖
   * 不用重畫整棵樹
